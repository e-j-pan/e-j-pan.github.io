WLED Remote Control Guide

Complete documentation for controlling your bedroom ceiling LED lights

About Your WLED Setup

This guide provides complete documentation for the Tuya TS004F Zigbee remote that controls the bedroom ceiling WLED lights. Whether you're a family member or guest, this guide will help you use the remote effectively.

System Components

  • Controller: Dig-Uno-V3 ESP32
  • LED Strip: 480 addressable RGB LEDs
  • Remote: Tuya TS004F Zigbee Remote
  • Hub: Zigbee Coordinator integrated with Home Assistant
  • Software: WLED v0.15.0

What is WLED?

WLED is an open-source software that controls LED strips through an ESP microcontroller. Our setup supports 187 different lighting effects and 72 color palettes, all controllable via the remote, smartphone app, or Home Assistant dashboard.

Tuya TS004F Remote Overview

The Tuya TS004F (_TZ3000_abrsvsou) is a rotating knob remote with multiple control functions. This guide explains all the functions available with this remote.

Button Controls

Action Function Description
Short Press Toggle Turn the light on or off
Long Press Colorloop Start a color cycling effect
Long Release Stop Effect Return to solid color

Rotary Controls

Action Function Description
Slow Right Turn Brightness Up Increase brightness by 10%
Slow Left Turn Brightness Down Decrease brightness by 10%
Fast Right Turn Next Effect Change to next WLED effect
Fast Left Turn Previous Effect Change to previous WLED effect
Right Rotation Warmer Increase color temperature
Left Rotation Cooler Decrease color temperature

Battery Replacement

The remote uses a CR2450 battery. When the battery level drops below 10%, it's time to replace it.

  1. Locate the small indent on the back of the remote
  2. Use a fingernail or small flat tool to gently pry open the back cover
  3. Remove the old CR2450 battery
  4. Insert the new CR2450 battery with the positive (+) side facing up
  5. Snap the back cover into place
  6. Test the remote to confirm it's working

Changing Effects

You can change effects by:

  • Fast-rotating the remote knob right (next) or left (previous)
  • Using the WLED app on your phone
  • Using the Home Assistant dashboard

Troubleshooting

Remote Not Responding

  1. Check if the battery is low (below 10%)
  2. Replace the CR2450 battery if needed
  3. Ensure the remote is within range of your Zigbee network
  4. Try restarting the remote by removing and reinserting the battery

Light Shows Wrong Colors

  1. Check if the right effect is selected
  2. Try changing to Solid effect, then back to your desired effect
  3. Restart the WLED controller by toggling power

Light Won't Turn On

  1. Check if the WLED controller has power
  2. Verify the WLED device is connected to your network
  3. Try restarting your Home Assistant server
  4. Check the automation is enabled in Home Assistant

Connection Issues

  1. Ensure your Zigbee network is functioning properly
  2. Check if the WLED device has a valid IP address
  3. Verify Home Assistant can communicate with the WLED device
